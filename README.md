# Capstone-Project-RealEstate-Insights-Gurgaon-ML-engineering-
Machine Learning & Core Pipeline Workflow
1. Data Acquisition & AggregationThe dataset was natively built by web scraping residential real estate listings from 99acres. Raw data collection culminated in two primary subsets: houses.csv and flats.csv. Merging these structural subsets yielded a unified database containing ~4,000 observations across 14 to 16 feature columns, establishing a robust baseline for market analysis.
2. Data Cleaning & Structural PreprocessingTo prepare the dataset for analysis, an aggressive data-cleansing pipeline was executed:Deduplication: Identified and eliminated duplicate rows to prevent downstream data redundancy.Structural Renaming: Standardized raw scraped column headers into clean, human-readable semantic variables.Initial Missing Value Management: Isolated and mapped the structural voids within the data.
3. Advanced Feature Engineering & Specialized ImputationDue to the messy nature of real-world real estate listings, resolving missing data and engineering granular attributes was a primary focus:Area Dimension Modeling: Property dimensions were split across three distinct categories: built_up_area, super_builtup_area, and carpet_area. Mathematical ratio constraints based on sector medians were derived to impute missing metrics across these interconnected dimensions.Categorical Imputation: High-volume missing records in agePossession and furnish_details were systematically resolved using a custom three-tier mode imputation strategy grouped by sector and property type.Unstructured Feature Extraction: Semi-structured "Features" and "Amenities" strings were parsed using a MultiLabelBinarizer to calculate an objective, weighted luxury_score for every listed property.
4. Statistical Outlier Detection & TreatmentTo protect model optimization from anomaly-driven bias, columns were evaluated based on their distribution properties:Parametric Features: Handled via standard Z-score thresholds where data distribution approximated normality.Non-Parametric Features: Evaluated via Interquartile Range (IQR) gating coupled with boxplot visualizations to truncate extreme values and unrepresentative price spikes safely.
